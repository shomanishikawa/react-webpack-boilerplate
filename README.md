# react-webpack-boilerplate
A pretty cool boilerplate. For Dev and Prod.
  * Node (Express)
  * Webpack (Babel, PostCSS, Hot Module Replacement)
  * React
  * Redux

## Dev
```
$ npm install
$ npm start
```

## Prod (Heroku)

```
heroku config:set NODE_ENV=production
git push heroku branch-name
```
